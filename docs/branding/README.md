# Logo Assets

Generated logo files for Dairy Management App:

- dairy-management-logo-primary.svg (horizontal primary logo)
- dairy-management-logo-mark.svg (square icon mark for app/store)

## Recommended use

- App icon / Play listing: use `dairy-management-logo-mark.svg` and export to 512x512 PNG.
- Website / docs header: use `dairy-management-logo-primary.svg`.

## Quick export options

- Figma: import SVG and export PNG.
- Inkscape: `inkscape dairy-management-logo-mark.svg --export-type=png --export-width=512 --export-filename=icon-512.png`

## Color palette

- Primary green: `#47685A`
- Dark text: `#2D4338`
- Accent leaf: `#6BAD4B`
- Light background: `#F4F8F5`
